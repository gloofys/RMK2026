## Challenge Summary

Create a probability scale from real-world events using reproducible code and a
clear visual presentation.

## Open Questions

- Which data sources give the most interesting probabilities?
- How should events be normalized so they are easy to compare?
- What chart design best communicates large differences in scale?

## Initial Plan

1. Pick one or two strong public datasets.
2. Extract event counts and denominators.
3. Convert events into probabilities on a consistent scale.
4. Prototype a readable visual layout.
